# Görev Takip Web App

Bu proje, Flask kullanılarak geliştirilmiş basit bir görev (to-do) uygulamasıdır.

## 🚀 Özellikler
- Görev ekleme
- Görev silme
- Görev arama
- Toplam görev sayısını gösterme

## 🛠 Kullanılan Teknolojiler
- Python
- Flask
- HTML
